Retail Store Sales Dashboard (Power BI)
📌 Overview

This project showcases an interactive Retail Sales Dashboard developed using Power BI. The dashboard is designed to transform raw sales data into meaningful insights, helping business owners monitor performance and make data-driven decisions.

The dataset used in this project is sourced from Kaggle and represents a sample retail store’s operations.

🎯 Objective

To build a dynamic and user-friendly dashboard that:

Tracks overall sales performance
Identifies profit trends and patterns
Highlights top-performing regions, products, and customers
Enables quick and efficient business analysis

🚀 Dashboard Highlights

🔎 Interactive Filters
Quarter-wise filtering (Q1, Q2, Q3, Q4)
State-wise selection for localized analysis

📊 Key Performance Indicators (KPIs)
Total Sales Revenue
Total Profit Earned
Total Quantity Sold

📈 Visual Insights
Monthly Profit Trends with profit/loss color indicators
Profit by Product Sub-Category for detailed analysis
Sales Distribution by Payment Mode
Category-wise Order Volume
Top 4 States contributing to sales
Top 4 Customers driving revenue

🔄 Smart Features
Reset button to quickly clear all filters (Ctrl + Click)
Clean and intuitive layout for better user experience

🧠 Key Insights You Can Derive
Seasonal trends in profit and sales
High-performing product categories
Customer purchasing behavior
Regional sales strengths and weaknesses

🛠️ Tools & Technologies
Power BI
Data Cleaning & Transformation
Data Visualization Techniques
